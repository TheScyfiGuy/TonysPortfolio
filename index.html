Project: Kingdom Builder App

About Project

The purpose of this project is to create a gaming aid web application for a popular tabletop game known as Dungeons and Dragons (D&D). This application will incorporate rules from an extension of the D&D ruleset, specifically the Kingdom Maker rules. Currently, there are no applications, web or mobile, that assist with this ruleset as there are for the base D&D rulesets. The Kingdom Maker or Kingdom Builder app will allow players to build and manage cities within the application according to the ruleset.

The project has been developed and designed in the GCU classes CST-451 and CST-452. With continual iterations and progress post class my hope is to make this site into a widely used app for all those D&D players out there. The approach to implimentation has been done with the Agile methodology. There will be code examples and a user/admin guide below.
